# sv

## Get status
```
SERVICE=
sv $SERVICE status
```

## Start a service
```
SERVICE=
sv $SERVICE up
sv $SERVICE start
```

## Stop a service
```
SERVICE=
sv $SERVICE down
sv $SERVICE stop
```

## Reload
```
SERVICE=
sv $SERVICE reload
```

## Restart
```
SERVICE=
sv $SERVICE restart
```

## Shutdown
```
SERVICE=
sv $SERVICE shutdown
```

## Check
Verify the service is in the requested state
```
SERVICE=
sv $SERVICE check
```
